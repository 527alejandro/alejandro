## Setup

### Prerequisites
- Install [Node.js]
- Install Node Package Manager

### Setting Up
Intall the Angular CLI globally:

```
npm install -g @angular/cli
```

Create workspace:
```
ng new [PROJECT NAME]
```

Run the application:

```
cd [PROJECT NAME]
ng serve
```
